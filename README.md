# 🎓 Student Life Assistant

## Overview

Student Life Assistant is a Python desktop application built using Tkinter to help students organize their daily academic tasks and improve productivity. The application allows users to manage tasks, assign priorities, estimate study duration, track progress, and generate productivity summaries through a simple and user-friendly interface.

Tasks are automatically saved locally using JSON, allowing users to continue where they left off each day.

---

## Features

* ➕ Add new tasks
* 📋 View all tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* ⭐ Assign High, Medium, or Low priority
* ⏱️ Estimate study duration for each task
* 📊 Generate daily productivity summaries
* 💡 Display productivity suggestions
* 💾 Automatically save tasks using JSON

---

## Technologies Used

* Python
* Tkinter (GUI)
* JSON (Local Data Storage)
