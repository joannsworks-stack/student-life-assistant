# 🎓 Student Life Assistant

## Overview

Student Life Assistant is a Python Tkinter desktop application designed to help students organize daily academic tasks and improve productivity.

The application allows users to add tasks, assign priorities, estimate study duration, mark tasks as completed, delete tasks and generate productivity summaries.

## Features

* Add tasks
* Assign High, Medium and Low priorities
* Estimate study duration
* View tasks
* Mark tasks as completed
* Delete tasks
* Generate productivity summaries
* Display simple productivity suggestions

## Technologies Used

* Python
* Tkinter
* JSON

## Setup Instructions

1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder.
4. Run:

python student_life_assistant_gui.py
